# Loan Data Analysis Dashboard

## Project Overview

This project focuses on analyzing loan-related data to better understand customer profiles, loan characteristics, and risk factors. The goal of this project was to clean and explore the dataset, identify useful patterns, and create an interactive Power BI dashboard to present the results in a clear and meaningful way.

The dataset includes information such as customer age, income, education level, employment experience, credit score, loan amount, loan purpose, and loan status.

---

## Objectives

* Clean and prepare the dataset for analysis
* Explore relationships between customer and loan variables
* Identify patterns and trends in loan data
* Analyze risk and credit-related factors
* Build an interactive Power BI dashboard

---

## Tools Used

* Power BI
* Python
* Pandas
* Matplotlib

---

## Work Completed

* Cleaned and prepared the dataset
* Performed exploratory data analysis (EDA)
* Created visualizations to understand data patterns
* Analyzed applicant characteristics and risk factors
* Built an interactive dashboard with filters and multiple pages

---

## Dashboard Preview

### Dashboard Page 1 – Loan Portfolio Overview

This page provides an overall summary of the loan dataset through key metrics such as total applications, total loan amount, default rate, and average interest rate. It also highlights loan amount distribution and default patterns across different loan purposes and customer categories.

---

### Dashboard Page 2 – Applicant Profile Analysis

This page focuses on applicant characteristics including age, income, education level, employment experience, and gender distribution. The analysis helps identify relationships between applicant profiles and loan-related factors.


---

### Dashboard Page 3 – Risk & Credit Analysis

This page examines risk-related indicators such as credit score, repayment behavior, previous default history, and loan-to-income ratio. The purpose is to understand factors that may influence loan default risk.

---

## Key Insights

* Loan applications were concentrated within specific age groups and income levels
* Applicant characteristics showed relationships with loan amount and repayment behavior
* Credit score and previous loan default history may affect default risk
* Loan purpose and financial background can influence loan performance
